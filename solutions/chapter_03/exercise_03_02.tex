For the given polynomial $p(x) = 3 x^2 + 2 x + 1$ we get the following results:

\begin{consoleoutput}
    >>> p = Polynomial([1,2,3])
    >>> p.derivative().coeff
    [2, 6]
    >>> p.antiderivative().coeff
    [0, 1.0, 1.0, 1.0]
    >>> p.antiderivative().derivative().coeff
    [1.0, 2.0, 3.0]
\end{consoleoutput}
